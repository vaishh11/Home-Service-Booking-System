# 🏠 Home Service Booking System

A full-stack MERN web application that enables users to book home services through a secure 
and user-friendly platform. The system provides service booking, payment management, booking tracking, feedback collection, 
and an admin dashboard for efficient service management.

---

## 🚀 Features

- User Registration & Login with JWT Authentication
- Browse and Book Home Services
- Secure Payment Management
- Track Booking Status
- User Profile Management
- Admin Dashboard
- Service & Category Management
- Booking & Payment Management
- Feedback & Ratings
- Responsive User Interface

## ⚙️ Installation

### Clone the repository

```bash
git clone https://github.com/vaishh11/Home-Service-Booking-System.git
```

### Navigate to project

```bash
cd Home-Service-Booking-System
```

### Install frontend dependencies

```bash
cd client
npm install
```

### Install backend dependencies

```bash
cd ../server
npm install
```

### Run frontend

```bash
cd client
npm run dev
```

### Run backend

```bash
cd ../server
npm start
```
